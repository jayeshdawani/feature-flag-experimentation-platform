# AI Ticket Triage Platform

An AI-assisted knowledge base and ticket triage platform for organizing support requests and suggesting useful responses.

## Problem
Teams waste time repeatedly answering similar questions and manually sorting support tickets.

## Target Users
- Student teams
- Internal support teams
- Small organizations

## Core Features
- Ticket creation
- Ticket status tracking
- Knowledge base articles
- AI-assisted ticket summaries
- Suggested responses

## Tech Stack
- Python
- FastAPI
- PostgreSQL
- React
- TypeScript
- TailwindCSS

## Current Status
Week 1 setup in progress.

## Roadmap
- Week 1: Repo and concept setup
- Week 2: Ticket and knowledge article CRUD
- Week 3: AI abstraction and mock summary feature
